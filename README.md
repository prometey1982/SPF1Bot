Простой чат бот написанный с помощью Deepseek, который отвечает на сообщения через LLM модели. Ему можно задавать системный промпт, чтобы бот лучше соответствовал стилю беседы.
Поддерживается контекстное окно. Также, можно задавать ID пользователей, на сообщения которых он будет всегда отвечать, даже если не указали имя бота в сообщении.
В конфигурационном файле задаётся списком пользователей, которым бот отвечает и которые могут перезагрузить конфиг бота.
Это нужно для применения изменений из конфигурационного файла без его перезагрузки.
